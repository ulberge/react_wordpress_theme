React WordPress Theme

- - DESCRIPTION - -

A WordPress-React Theme

- Setup project -
npm install && bower install && grunt build

- Setup React project -
cd react/whiteboard-app && npm install && npm run build

- To develop on WordPress theme -
cd react/whiteboard-app && npm run build && cd ../.. && grunt

- To develop on React app as standalone page -
cd react/whiteboard-app && npm start

- To build for prod -
grunt build && cd react/whiteboard-app && npm run build && cd ../..
